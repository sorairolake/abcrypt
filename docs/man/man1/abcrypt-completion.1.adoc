== NAME

abcrypt-completion - generate shell completion

== SYNOPSIS

*abcrypt completion* _SHELL_

== DESCRIPTION

This command generates shell completion. The completion is output to standard
output.

== POSITIONAL ARGUMENTS

_SHELL_::

  Shell to generate completion for.

  The possible values are:{blank}:::

    *bash*::::

      Bash.

    *elvish*::::

      Elvish.

    *fish*::::

      fish.

    *nushell*::::

      Nushell.

    *powershell*::::

      PowerShell.

    *zsh*::::

      Zsh.
