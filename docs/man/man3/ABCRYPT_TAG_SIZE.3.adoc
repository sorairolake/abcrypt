== NAME

ABCRYPT_TAG_SIZE - API constant

== SYNOPSIS

[source,c]
----
#include <abcrypt.h>

#define ABCRYPT_TAG_SIZE 16
----

== DESCRIPTION

This constant represents the number of bytes of the MAC (authentication tag) of
the ciphertext.

== VERSIONS

This constant was added in version 0.1.0.
