== NAME

abcrypt_params_new - API function

== SYNOPSIS

[source,c]
----
#include <abcrypt.h>

struct abcrypt_params *abcrypt_params_new(void);

== DESCRIPTION

This function creates a new Argon2 parameters structure.

=== Arguments

Takes no arguments.

== RETURN VALUE

Returns a pointer to the Argon2 parameters structure.
