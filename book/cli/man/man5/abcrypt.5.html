NAME

abcrypt - abcrypt encrypted data format

DESCRIPTION

abcrypt is a modern file encryption format with the data authenticity inspired by the scrypt encrypted data format.

It uses Argon2id for key derivation, BLAKE2b-512-MAC for header integrity checking and XChaCha20-Poly1305 for encryption.

File format structure

An abcrypt file is composed of two parts: a header containing the required data and a file body encrypted with the derived key.

Filename

abcrypt files may use the extension .abcrypt.
