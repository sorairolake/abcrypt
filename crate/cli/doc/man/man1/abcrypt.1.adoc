== NAME

abcrypt - an utility for encrypt and decrypt files

== SYNOPSIS

*{manname}* [_OPTION_]... _COMMAND_

== DESCRIPTION

*{manname}* is a command-line utility for encrypt and decrypt files.

== COMMANDS

*abcrypt-encrypt*(1)::

  Encrypt files.

*abcrypt-decrypt*(1)::

  Decrypt files.

*abcrypt-information*(1)::

  Provides information about the encryption parameters.

*abcrypt-help*(1)::

  Print the help for a command.

== OPTIONS

*-h*, *--help*::

  Print help information.

*-V*, *--version*::

  Print version information.

*--generate-completion* _SHELL_::

  Generate shell completion. The completion is output to stdout.

  The possible values are:{blank}:::

    *bash*::::

      Bash.

    *elvish*::::

      Elvish.

    *fish*::::

      fish.

    *nushell*::::

      Nushell.

    *powershell*::::

      PowerShell.

    *zsh*::::

      Zsh.
